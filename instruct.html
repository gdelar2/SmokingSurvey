Instructions for survey app by: Gus De La Rosa Michael Brooks Kevin Katsis


Step 1: Download and Install Eclipse Java IDE from: http://www.eclipse.org/downloads/


Step 2: In Eclipse install the Android plugin

a. Start Eclipse, then select Help > Install New Software.

b. Click Add, in the top-right corner.

c. In the Add Repository dialog that appears, enter "ADT Plugin" for the Name and the following URL for the Location: https://dl- ssl.google.com/android/eclipse/

d. Click OK. If you have trouble acquiring the plugin, try using "http" in the Location URL, instead of "https" (https is preferred for security reasons).

e. In the Available Software dialog, select the checkbox next to Developer Tools and click Next.

f. In the next window, you'll see a list of the tools to be downloaded. Click Next.

g. Read and accept the license agreements, then click Finish.

h. If you get a security warning saying that the authenticity or validity of the software can't be established, click OK.

i. When the installation completes, restart Eclipse.


Step 3: Create an android emulator. We used API 10 because one of us had an older phone.


Step 4: Create an android project.


Step 5: In the first page enter the following: Application name: Smoking Survey Project name: Smoking Survey Package name: com.CS340.smokingsurvey Build sdk: API 10


Step 6: Click next 3 times, under "new Blank activity" enter: HelloAndroidActivity. Then click finish.


Step 7: Drag the files from our repo into the following directory in eclipse SmokingSurvey |--src |--com.CS340.smokingsurvey


Step 8: In the prompt that appears click "LINK" relative to project_LOC


Step 9: outside of eclipse drag the testing.txt file into the assests folder.


Step 10: click run and select the correct AVI to run the program.
